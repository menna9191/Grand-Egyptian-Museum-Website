# Grand Egyptian Museum Website

## Overview

The Grand Egyptian Museum Website is a responsive web application designed to showcase the Grand Egyptian Museum, its exhibits, visitor information, and booking services. The website provides an engaging user experience through modern design, interactive sections, and responsive layouts.

## Features

* Responsive design for desktop, tablet, and mobile devices
* Interactive navigation menu
* Museum information and history section
* Exhibits and artifacts showcase
* Image gallery
* Visitor booking section
* Location and map section
* Contact information
* Smooth user experience with dynamic JavaScript interactions

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Installation

1. Clone the repository:

```bash
git clone https://github.com/menna9191/Grand-Egyptian-Museum-Website.git
```

2. Navigate to the project directory:

```bash
cd Grand-Egyptian-Museum-Website
```

3. Open `index.html` in your browser.
